Шаг 3: СБОР ПОДПИСЕЙ/ОПРОС В МЕССЕНДЖЕРАХ ПО ОПРЕДЕЛЕНИЮ НАИБОЛЕЕ АКТУАЛЬНОГО ИНИЦИАТИВНОГО ПРОЕКТА
Организация проведения предварительных процедур
Решение о проведении сбора подписей в поддержку инициативного проекта, выдвигаемого для внесения на рассмотрение в местную администрацию, принимается на собрании инициаторов. Рекомендуем данное решение внести в протокол собрания инициаторов проекта.
Данное мероприятие представляет собой подворовый обход/опрос в мессенджерах и проводится инициаторами инициативного проекта самостоятельно до внесения проекта в местную администрацию.
Форма подписного листа для сбора подписей в поддержку инициативного проекта
После окончания сбора подписей/опроса в мессенджерах инициаторы проекта прилагают заверенные подписные листы/скриншоты к протоколу собрания инициаторов проекта и передают вместе с пакетом документов для внесения проекта в администрацию.
Инициаторы проекта могу обратиться в местную администрацию с обращением об оказании содействия в организации мероприятия по сбору подписей/опроса в мессенджерах по вопросу о поддержке инициативного проекта.

Администрация МО совместно с инициаторами проекта публикует информацию о проведении мероприятия общественного обсуждения инициативного проекта на своих официальных информационных ресурсах.
Укажите в протоколе итогового собрания граждан информацию о результатах общественного обсуждения. Обязательно должно быть указано: наименование инициативного проекта и количество человек, которые поддержали данную проектную идею. Например:
Сбор подписей/опрос в мессенджерах граждан по вопросу о поддержке инициативного проекта осуществляется инициаторами проекта посредством внесения подписей граждан в подписной лист или организации электронного голосования на специализированных платформах.

При организации мероприятия по сбору подписей/опроса в мессенджерах в поддержку инициативного проекта следует придерживаться следующих правил:
3
Данные о гражданине, его подпись и дата внесения в подписной лист вносятся гражданином собственноручно.
4
Каждый подписной лист с подписями граждан должен быть заверен инициаторами проекта.
2
В сборе подписей вправе принимать участие жители соответствующей территории, достигшие
18-летнего возраста.
1
Участие в сборе подписей граждан осуществляется добровольно и основано на принципах законности, гласности, открытости.
Данные вносятся рукописно (использование карандаша недопустимо). Гражданин вправе ставить подпись в поддержку одного и того же инициативного проекта только один раз.

Рекомендуем не указывать в документе дополнительные персональные данные респондентов, чтобы исключить необходимость получения отдельного согласия на обработку персональных данных от каждого участника опроса.
В случае если инициатором проекта выступает инициативная группа, подписной лист заверяется любым членом инициативной группы.
Как сделать так, чтобы начислили баллы?
В рамках изучения общественного мнения по вопросу поддержки инициативного проекта по приобретению детского игрового оборудования инициативной группой был проведен сбор подписей. По результатам проделанной работы проект поддержали 250 человек.
